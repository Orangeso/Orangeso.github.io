快捷键

发表于 2018-01-19 | 分类于 吼 |

快捷键

⌘⇧P   显示代码预览
⌘6  导出
⌘8  标签表
快捷键中常用符号⌘（command）、⌥（option）、⇧（shift）、⇪（caps lock）、⌃（control）、↩（return）、⌅（enter）。
command+control+空格 显示符号

基本操作

  • ⌘N 创建新文稿
  • ⌘← 将文稿移至废纸篓
  • ⇧⌘B 在当前光标位置分割文稿
  • ⇧⌘N 创建新组
  • ⌃⌘N 创建新过滤器
  • 项目总是会紧邻当前选择的项目创建。
  • 文稿会被创建在当前所选的下面。
  • 组和过滤器会被创建在当前所选的组中。
  • 如果没有选择，或者这个选项不接受新的内容，项目将建立在最顶层类别的“收件箱”中（例如 iCloud 中）。
  • 点击进行选择
  • 双击进行编辑

视图和常用功能

  • ⌘1 切换文库
  • ⌘2 切换文稿列表
  • ⌘3 切换仅编辑器视图
  • ⌘4 切换“附件栏”
  • ⌘5 前往“喜爱”组 (FIVEorites)
  • ⌘6 显示“快速导出”(SIXport)
  • ⌘7 显示“统计”
  • ⌘8 显示导航器（NavigEIGHTor，获得它？）
  • ⌘9 显示“标记栏”

“快速打开”面板

  • ⌘O 在整个文库中查找文稿
  • ⇧⌘O 当前文库部分（iCloud、在我的 Mac 上等）中查找文稿
  • ⌥⌘O 在当前选定的组中查找文稿

全文搜索

  • ⇧⌘F 将焦点切换至文稿过滤器
  • esc 离开文稿过滤器

编辑器中的“查找和替换”

  • ⌘F 显示“查找”面板
  • ⌥⌘F 显示“查找和替换”面板
  • esc 关闭“查找/替换”面板

喜爱

  • ⌘D 将文稿添加到喜爱或者将它从喜爱中移除
  • 您也可以使用文稿的上下文菜单进行该操作。
  • ⌘5 前往“喜爱”组（仅在有喜爱的文稿时才显示）

编辑器智能粘贴

  • ⇧⌘V 调用“智能粘贴”或
  • 按住 ⌘V 调用“智能粘贴”：
    • “粘贴”选项随剪贴板内容和当前标记而变。
    • 左/右剪头键预览可用到粘贴选项
    • ↩︎ 确认并粘贴

屏幕模式

  • ⌥⌘X 切换深色主题
  • ⌃⌘F 切换全屏模式
  • ⌥⌘L 切换黑暗模式

多个窗口，Split View

  • 您可以同时打开多个窗口（⌥⌘N）
  • 我们支持 MacOS 的 Split View，所以您可以在 Split View 中显示两个靠在一起的窗口。
  • 您还可以在 Split View 中的另一个应用旁边或者在其自己的预览旁边显示 Ulysses 文稿。
  • 您甚至可以打开所有的窗口，这使得您在编辑时显得很热闹，说实话，这真的没有什么多大用处。

标签

  • ⇧⌘T 切换标签栏
  • ⌘T 打开新标签
  • ⌃⇥ 切换到下一个标签
  • ⌃⇧⇥ 切换到上一个标签
  • 将标签拖出使之成为一个单独的窗口。
  • 将所有打开的窗口放入一个窗口的的标签中，从“窗口”菜单中选择“合并所有窗口”。

文稿列表选项

  • “视图”菜单有几个自定义文稿列表的选项：
    • 将“文稿预览”设置为 1 行至 6 行之间的任意值。
    • 在文稿预览中包含关键词和目标。
    • 按日期或首字符手动排序。

编辑器选项

  • “视图”菜单还有几个自定义编辑器的选项：
    • ⌥⌘T 切换“打字机模式”
    • 启用显示段落号

字体大小

  • ⌘+ 增加字体大小
  • ⌘+ 减小字体大小
  • ⌘0 将字体大小重置为默认值

标记

  • ⌘\\ 用于标题
  • ⌘I 强调
  • ⌘B 粗体
  • ⌘K 链接
  • ⌘L 移除任何类型的标记
  • ⌘9 显示标记栏，以防您忘记任何标记

智能标签

  • 如果启用，只要一个开始标签后跟一个字符，就会自动插入结束标记占位符。
  • 通过“编辑” › “替换”来启用智能标签。

结束标记完成

  • 按esc自动输入结束标记占位符。
  • 接受结束标记，做以下任何一种：
    • 按 ↩︎
    • 按 →
    • 按 ↓（如果是最后一张文稿）
    • 键入结束标记
  • 要消除结束标记，按 esc。

转义符

  • 为了防止 Ulysses 将任何字符当成标记，只需在它前面添加一个反斜杠（例如，若您真想要方括号而不是链接，则输入 “[内容]”）。

标记栏

  • 显示“标记栏”(⌘9) 时，您可以点按列出的任何标记，将其应用到当前选择的项目。
  • 这适用于多个选择（⌘-点按选择文本的不同部分）。
  • 多层次的段落，如列表和嵌段引号，在您每次点按时缩进+1。
  • ⇧-点按可移除标记或凸排多层次的段落。

标记语言

  • 默认情况下，Ulysses 使用其原生标记语言“Markdown XL”。
  • 通过“编辑” › “转换标记”切换至“Markdown”、“Textile’d”或“Minimark”。
  • 通过“偏好设置” › “标记”，您可以创建自己的标记。

列表

  • ⇥ 增加列表缩排
  • ⇧⇥ 减小列表缩排

智能列表

  • 如果启用，按 ↩︎ 会自动添加另一个列表项，而按 ⌥↩︎ 则只会创建一个新段落。
  • 通过“编辑” › “替换”来启用智能列表。

媒体

  • 拖放图像和 PDF 文件到编辑器中来创建一个拥有这些内容的文本对象。

文本对象

  • ⌘-点击选择
  • 点按两下打开文本对象弹跳窗

文本对象弹跳窗

  • ⌘↩︎ 关闭并向前移动插入点
  • esc 关闭并保持对象选定
  • ⇥ 在输入字段中切换
  • 拖动任一弹跳窗将其撕开，然后将它放在屏幕上任何地方。

导航，文库

  • → 作用于组，则打开组
  • ⌘→ 作用于组，则将焦点切换至文稿列表

导航，文稿列表

  • → 将焦点切换至编辑器中（将插入点置于最后编辑的位置）
  • ← 将焦点切换至文库中

导航、编辑器

  • 在文稿开始时：
    • 轻点两下 ↑ 向上移动一个文稿。
  • 在文稿结束时：
    • 轻点两下 ↓ 向下移动一个文稿。
  • 在表内任何地方：
    • ⌥⌘↑ 移动至上一个文稿
    • ⌥⌘↓ 移动至下一个文稿
  • 轻点两下 ⌘← 将焦点切换至文稿列表中。

复制和移动的侧边栏

  • 在默认情况下，将组和文稿拖入和拖出 iCloud 或“在我的 Mac 上”将总是移动，也就是说内容将转移到新的位置。
  • 将内容拖入和拖出外部文件夹将总是拷贝，也即就说，执行操作后内容将同时存在于两个位置。
  • 按住 ⌥ 同时进行拖动，强制拷贝组和文稿。
  • 按住 ⌘ 同时进行拖动，强制移动组和文稿。

快速导出

  • ⌘6 针对当前编辑器显示“快速导出”(SIXport)
  • 您还可以右键点按侧边栏中的组、过滤器或文稿，然后选择“快速导出…”。

文本导出

  • 在纯文本、富文本、Markdown 和 TextBundle 输出之间选择：
    • “纯文本”将标记、评论、批注、图像等全部去除，仅导出文本本身。
    • “富文本”将生成简单的 RTF 文件。
    • “Markdown”将保留（或转换）所有标记，按标记符语法导出文本。
    • “TextBundle” 是一种新的文件格式，支持在纯文本文件中嵌入图像。

HTML 导出

  • 选择输出时要使用到输出样式。
  • 在片段和单页输出中间选择：
    • “片段”将仅将所有标记转换为 HTML，然后导出文本。
    • “单页”将创建完整到 HTML 文稿，包括标题、正文、CSS 链接等。

ePub 导出

  • 选择输出时要使用到输出样式。
  • 您可以设置标题和作者元数据。
  • 您可以放入要使用的图像作为电子书的封面图像。

PDF 导出

  • 选择输出时要使用到输出样式。
  • 选择纸张格式。

DOCX 导出

  • 选择输出时要使用到输出样式。
  • 选择纸张格式。

发布

  • Ulysses 让您可以将所编辑的内容直接发布到 WordPress 博客：
    • 选择“添加帐户 › WordPress/定制 WordPress”。
    • 提供您的用户名和密码，对于自定义 WordPress，还需要输入 URL。
    • 发布文章前，您可以修改其设置（类别、标签、日程安排、发布状态等。）
  • 如果您拥有 Medium 帐户，那么 Ulysses 可以让您将文本作为草稿发布到 Medium 上：
    • 选择“添加帐户” › “Medium”。
    • 您需要提供 integration token 进行激活。

导出预览

  • ⇧⌘P 预览当前侧边栏选择（可以是组）
  • ⌘R 刷新所有打开的预览
  • 选择用来预览的导出样式。
  • 您可以直接从预览窗口导出。

打开并导出文件

  • 您可以使用 Ulysses 编辑或分别导入一些不同的文件类型：
    • 默认情况下，文本和 Markdown 文件将出现在文库的“打开文件”组中。所有编辑和更改将保存到原文件中。
    • 要导出文本和 Markdown 文件作为 Ulysses 文稿，只需将它们拖动到文库内的组中。原文件将不受任何编辑的影响。
    • RTF、DOCX、ENEX 和 HTML 文件将总是导入为 Ulysses 文稿，因此将添加到最顶端文库部分的“收件箱”。当然了，您可以随意拖动它们。

其他

  • ⌘S 将在当前工作表中创建一个新版本。从“文件”菜单上选择“浏览所有版本…”来看看您多久调用一次这个指令。
  • 双击文稿列表中的文稿可以让您跳转到上次编辑的位置。
